What I Build and Ship.

Two tracks running in parallel: a three-project infrastructure trilogy building production-grade AWS security from the ground up, and a series of Go web apps built, secured, and deployed to production.

infrastructure trilogy

Three interconnected projects. Each one builds on the last. The goal: a fully automated, tested, and monitored cloud security infrastructure - provisioned as code, hardened, and deployed through a CI/CD pipeline that enforces quality gates before anything ships.

PROJECT 01 · WEEKS 1-5
Secure Foundation
Terraform-First AWS
Security Infrastructure

Before writing a Lambda function or wiring up a detection pipeline, I needed to prove I could build the infrastructure underneath it correctly - provisioned as code, tested, linted, and documented. This is the secure AWS foundation that everything else runs on: VPC, IAM, S3 remote state, CloudTrail, GuardDuty, and AWS Config - all Terraform, no console clicks.

PROJECT 02 · WEEKS 6-10
Hardened by Default
Containerized Security
Monitoring on EKS

Infrastructure without visibility is just hope. Project 2 deploys a full observability stack on EKS - Grafana, Prometheus, and a custom API exposing security metrics from the detection pipeline - all provisioned with Terraform on top of Project 1's foundation, with EC2 nodes hardened to CIS benchmarks via Ansible. One dashboard. Full visibility.

PROJECT 03 · WEEKS 11-14
Shift Left
Tested, Automated
Infrastructure CI/CD Pipeline

Projects 1 and 2 were run manually. That's not how production teams operate. Project 3 builds the pipeline that enforces quality gates before anything ships: tflint, Checkov, and Terratest on the infrastructure side; Trivy container scanning and Ansible hardening on the application side. Nothing deploys unless it passes.

go web apps — 10 apps series

Ten production web applications written in Go. Each one built from scratch, secured, containerized, and shipped to AWS. The goal is full-stack fluency in Go - not just backend concepts, but building, hardening, and operating real applications end to end.

APP 01
Personal Uptime Monitor

HTTP pinger that checks 6 URLs every 30 seconds using goroutines. Persists results in SQLite, exposes a REST API, serves a live dashboard, and exports Prometheus metrics to Grafana.
