About Kumo Security

Kumo Security builds cloud security tooling for SaaS startups. The first product is Kumo Assess — an agentic SOC 2 compliance assessment that connects to your AWS account and runs a full control gap analysis in hours, not months.

Background

  • Cloud and infrastructure security — AWS, Terraform, Ansible, Docker, Kubernetes
  • Security engineering — container security, WAF, TLS/PKI, authentication and authorization
  • Compliance frameworks — SOC 2, ISO 27001, CMMC, GDPR in cloud environments
  • Observability and CI/CD — Grafana, Prometheus, GitHub Actions
  • Languages — Go, TypeScript; AI tooling built on Anthropic/Claude

What We're Building

Kumo Assess is an agentic compliance system. Claude agents collect evidence from your AWS environment, score each SOC 2 control, and generate specific remediation guidance — all in a single guided session. Currently covers SOC 2 CC6 (Logical Access) and CC7 (System Operations).

Kumo Assess is not a checkbox tool. It's built on direct AWS API calls, not heuristics. The assessment logic is transparent and inspectable.
